## docker run

```bash
docker run --name rrshare \
  -d -p3001:3001 \
  -v ~/Downloads/rrshare:/opt/work/store:rw \
  gowah/rrshare64:latest
```

## 异常处理

rrshare客户端做的好不太完善，最简单的删除重建

```bash
docker rm -f rrshare
```
